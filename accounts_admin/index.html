Comprehensive Accounting(CA)-Administration Panel

কিভাবে সফটওয়্যারে লগইন করবেন

Comprehensive Accounting সফটওয়্যারের Admin Panel এ লগইন করার জন্য প্রথমে আপনার কম্পিউটারের ইন্টারনেট কানেকশন নিশ্চিত করতে হবে। তারপর ইন্টারনেট ব্রাউজার ওপেন করতে হবে। সফটওয়্যারটির Recommended ইন্টারনেট ব্রাউজার হচ্ছে Google Chrome এবং Mozilla Firefox. তবে Updated যেকোনো Browser দ্বারা এই Software ব্যাবহার করা যাবে।

ইন্টারনেট ব্রাউজার ওপেন করার পর ব্রাউজার এর Address Bar এ সফটওয়্যারের Login link (https://demo.caccounts.com/admin) লিখতে হবে।

Login link লেখার পর কী-বোর্ডের ইন্টার কী প্রেস করতে হবে,প্রেস করার পর উক্ত Software এ প্রবেশ করার জন্য একটি Login Form আসবে।


লগইন ফর্মে দুইটি টেক্সট বক্স আছে, একটি Username এবং অপরটি Password। উক্ত টেক্সট বক্সে ইউজারনেইম ও পাসওয়ার্ড দেওয়ার পর লগইন বাটন এ ক্লিক করতে হবে।ইউজারনেইমও পাসওয়ার্ড যদি বৈধ হয় তা হলে সফটওয়্যারটি এক্সেস করার অনুমতি পাবেন। আর যদি বৈধ না হয় তাহলে পুনরায় লগইন ফর্মে ফেরত পাঠাবে।

আপনি সফলভাবে Software এ Login করার পর নিচের Page টি দেখতে পাবেন।

User Role

User Role: User Role এ Click করলে আপনি Existing "User Role" এর List দেখতে পাবেন। এখান থেকে Search Box এ আপনার প্রয়োজনানুসারে Search করেও কাঙ্ক্ষিত User Role টি দেখতে পারবেন।


চিত্রে পেইজটি দেখানো হলঃ



নতুন User Role Add করার নিয়মঃ

আপনাকে User Role থেকে Add new বাটনে Click করতে হবে এবং User Role নামের একটি Form Open হবে। তারপর সকল Field এর তথ্য সঠিক ভাবে Select/Fillup করার পর Save বাটনে Click করলে নতুন User Role টি Add হয়ে যাবে এবং সেটি আপনি User Role List এ দেখতে পাবেন।

আপনাকে Sub Ledger Group থেকে Add new বাটনে Click করতে হবে এবং Sub Ledger Group নামের একটি Form Open হবে। তারপর সকল Field এর তথ্য সঠিক ভাবে Select/Fillup করার পর Save বাটনে Click করলে নতুন Sub Ledger Group টি Add হয়ে যাবে এবং সেটি আপনি Sub Ledger Group List এ দেখতে পাবেন।

Sub Ledger Group এর তথ্য Edit করার নিয়মঃ

Existing Sub Ledger Group এর তথ্য Edit করার জন্য আপানাকে Sub Ledger Group List থেকে কাঙ্ক্ষিত Sub Ledger Group এর ডান দিকে Edit Button (পেন্সিলের মতো) এ Click করতে হবে। তখন পূর্বের তথ্যাবলী সহ একটি Form Open হবে। প্রয়োজনীয় তথ্যাদি Correction করার পর Save বাটনে Click করলে সংশোধিত Sub Ledger Group এর তথ্য টি Sub Ledger Group List এ দেখতে পাবেন।

Sub Ledger Group Delete করার নিয়মঃ

Existing Sub Ledger Group Delete করার জন্য আপনাকে Sub Ledger Group List থেকে কাঙ্ক্ষিত Sub Ledger Group এর ডান দিকে Delete Button এ Click করতে হবে। তখন আপনাকে একটি নিশ্চিতকরণ মেসেজ দেখানো হবে, যেখানে আপনাকে জিজ্ঞাসা করা হবে “আপনি কি নিশ্চিত? এটি Delete করতে চান? আপনি এটি Delete করলে পুনরায় ফিরে পাবেন না!” এখন আপনি যদি নিশ্চিত হয়ে Delete করতে চান তাহলে Delete বাটনে Click করবেন। অন্যথায় Close বাটনে Click করে আগের মেনুতে ফিরে আসবেন।

Project Wise Group: Sub Ledger Group থেকে Project Wise Group এ Click করলে আপনি Existing Project এর Accounts সমূহের Access দিতে পারবেন অথবা Access Remove করতে পাবেন।

Depreciation Rate

Depreciation Rate: Config থেকে Depreciation Rate এ Click করলে আপনি General Ledger সমূহের Existing Depreciation Rate দেখতে পাবেন। আপনি এখান থেকে আপনার প্রয়োজনানুসারে General Ledger এর বিপরীতে Depreciation Rate পরিবর্তন করে Save করতে পারবেন।

Depreciation Code

Depreciation Code: Config থেকে Depreciation Code এ Click করলে আপনি Fixed Asset সমূহের Existing Depreciation Code এবং Accumulated Depreciation Code দেখতে পাবেন। আপনি এখান থেকে আপনার প্রয়োজনানুসারে Fixed Asset সমূহের বিপরীতে Depreciation Code এবং Accumulated Depreciation Code পরিবর্তন করে Save করতে পারবেন।

General Configuration

General Configuration: Config থেকে General Configuration এ Click করলে আপনি Report এবং Budget সমূহের Fraction এর Existing Status দেখতে পাবেন। আপনি এখান থেকে আপনার প্রয়োজনানুসারে Report এবং Budget সমূহের Fraction On/Off করে Save করতে পারবেন।
